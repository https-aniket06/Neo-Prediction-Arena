## ğŸš€ Getting Started

### Prerequisites
* Node.js (v18 or higher)
* A Firebase account and service account key
* API Keys for AI & Financial data providers

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/https-aniket06/Neo-Prediction-Arena.git
   cd Neo-Prediction-Arena
   ```

2. **Backend Setup:**
   Navigate to the backend directory and install dependencies:
   ```bash
   cd stitch_prediction_arena/backend
   npm install
   ```
   *Make sure to add your `.env` and `serviceAccountKey.json` files here securely.*
   ```bash
   npm run dev
   ```

3. **Frontend Setup:**
   Run the frontend by using a local development server (like VS Code's "Live Server" extension) at the project root to serve the `index.html` file in `frontend/`.
